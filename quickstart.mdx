---
title: "Quickstart: run your first multitaper spectrogram"
description: "Install dependencies, copy the Python implementation, and run your first multitaper spectrogram on a chirp signal in under five minutes."
---

This guide walks you through installing the Python implementation of the Multitaper Toolbox and running a complete working example. Tabs at the end of each relevant step also show the equivalent MATLAB and R code.

<Steps>
  <Step title="Install Python dependencies">
    Install all required packages with pip:

    ```bash
    pip install numpy scipy matplotlib joblib colorcet
    ```

    <Note>
      `colorcet` is a required import in `multitaper_spectrogram_python.py` — the file will fail to load without it. The `requirements.txt` in the `python/` directory lists `joblib`, `matplotlib`, `numpy`, and `scipy`; add `colorcet` alongside those.
    </Note>
  </Step>

  <Step title="Get the implementation file">
    The toolbox is a single self-contained file. Download or copy `multitaper_spectrogram_python.py` from the repository into your working directory:

    ```bash
    curl -O https://raw.githubusercontent.com/preraulab/multitaper_toolbox/master/python/multitaper_spectrogram_python.py
    ```

    Place the file in the same directory as your script, or add its location to your Python path.
  </Step>

  <Step title="Run the chirp example">
    The following example generates a 10-minute logarithmic chirp signal sweeping from 1 Hz to 20 Hz, then computes and plots the multitaper spectrogram.

    <Tabs>
      <Tab title="Python">
        ```python
        from multitaper_spectrogram_python import multitaper_spectrogram
        import numpy as np
        from scipy.signal import chirp

        # Set spectrogram params
        fs = 200                  # Sampling frequency (Hz)
        frequency_range = [0, 25] # Limit frequencies from 0 to 25 Hz
        time_bandwidth = 3        # Set time-half bandwidth
        num_tapers = 5            # Set number of tapers (optimal is time_bandwidth*2 - 1)
        window_params = [4, 1]    # Window size is 4s with step size of 1s
        min_nfft = 0              # No minimum nfft
        detrend_opt = 'constant'  # Detrend each window by subtracting the average
        multiprocess = True       # Use multiprocessing
        n_jobs = 3                # Use 3 cores in multiprocessing
        weighting = 'unity'       # Weight each taper at 1
        plot_on = True            # Plot spectrogram
        return_fig = False        # Do not return plotted spectrogram
        clim_scale = False        # Do not auto-scale colormap
        verbose = True            # Print extra info
        xyflip = False            # Do not transpose spect output matrix

        # Generate sample chirp data
        t = np.arange(1/fs, 600, 1/fs)  # 10-min time array from 1/fs to 600 stepping by 1/fs
        f_start = 1                       # Chirp freq range min (Hz)
        f_end = 20                        # Chirp freq range max (Hz)
        data = chirp(t, f_start, t[-1], f_end, 'logarithmic')

        # Compute the multitaper spectrogram
        spect, stimes, sfreqs = multitaper_spectrogram(data, fs, frequency_range, time_bandwidth, num_tapers, window_params, min_nfft, detrend_opt, multiprocess, n_jobs, weighting, plot_on, return_fig, clim_scale, verbose, xyflip)
        ```
      </Tab>
      <Tab title="MATLAB">
        ```matlab
        Fs = 200;                  % Sampling frequency
        frequency_range = [0 25];  % Limit frequencies from 0 to 25 Hz
        taper_params = [3 5];      % Time bandwidth and number of tapers
        window_params = [4 1];     % Window size is 4s with step size of 1s
        min_nfft = 0;              % No minimum nfft
        detrend_opt = 'constant';  % Detrend each window by subtracting the average
        weighting = 'unity';       % Weight each taper at 1
        plot_on = true;            % Plot spectrogram
        verbose = true;            % Print extra info

        % Generate sample chirp data
        t = 1/Fs:1/Fs:600;         % Create 10 minutes of data
        f_start = 1; f_end = 20;   % Set chirp range in Hz
        data = chirp(t, f_start, t(end), f_end, 'logarithmic');

        % Compute the multitaper spectrogram
        [spect, stimes, sfreqs] = multitaper_spectrogram(data, Fs, frequency_range, taper_params, window_params, min_nfft, detrend_opt, weighting, plot_on, verbose);
        ```
      </Tab>
      <Tab title="R">
        ```r
        install.packages("signal")
        library(signal)  # get signal library for chirp function

        # Set spectrogram params
        fs = 200                      # Sampling frequency
        frequency_range = c(0, 25)    # Limit frequencies from 0 to 25 Hz
        time_bandwidth = 3            # Set time-half bandwidth
        num_tapers = 5                # Set number of tapers (optimal is time_bandwidth*2 - 1)
        window_params = c(4, 1)       # Window size is 4s with step size of 1s
        min_nfft = 0                  # No minimum nfft
        weighting = 'unity'           # Weight each taper at 1
        detrend_opt = 'constant'      # Detrend each window by subtracting the average
        parallel = TRUE               # Use multiprocessing
        num_workers = 3               # Use 3 cores in multiprocessing
        plot_on = TRUE                # Plot spectrogram
        verbose = TRUE                # Print extra info
        xyflip = FALSE                # Do not transpose spect output matrix

        # Generate sample chirp data
        t = seq(1/fs, 600, by=1/fs)  # 10-min time array from 1/fs to 600 stepping by 1/fs
        f_start = 1                   # Set chirp freq range min (Hz)
        f_end = 20                    # Set chirp freq range max (Hz)
        data = chirp(t, f_start, tail(t, n=1), f_end, 'logarithmic')

        # Compute the multitaper spectrogram
        results = multitaper_spectrogram_R(data, fs, frequency_range, time_bandwidth, num_tapers, window_params, min_nfft, weighting, detrend_opt, parallel, num_workers, plot_on, verbose, xyflip)
        spect = results[[1]]
        stimes = results[[2]]
        sfreqs = results[[3]]
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Understand the return values">
    The function returns three arrays:

    | Return value | Shape | Description |
    |---|---|---|
    | `spect` | F × T matrix | Power at each time-frequency point. Rows are frequency bins, columns are time windows. |
    | `stimes` | 1D array | Center time of each window in seconds. Length T. |
    | `sfreqs` | 1D array | Frequency of each bin in Hz. Length F, spanning `frequency_range`. |

    In the chirp example with the parameters above, `spect` will have approximately 101 rows (frequency bins from 0 to 25 Hz) and 597 columns (time windows).

    <Tip>
      The optimal number of tapers for a given time-half bandwidth product is `floor(2 * time_bandwidth) - 1`. With `time_bandwidth = 3`, the optimal number of tapers is 5 — which matches `num_tapers = 5` in the example above.
    </Tip>
  </Step>
</Steps>
